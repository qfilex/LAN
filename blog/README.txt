How to Setup the blog
=====================
1.Create you database with two tables (posts, comments) and then import the sample data:
	-for post sample data import from ( projectfolder\sqlscript\posts.sql )
	-for post sample data import from ( projectfolder\sqlscript\comments.sql )

2.Change the connection string in ( projectfolder\core\Config.php )

3.Visit the site blog and enjoy.
